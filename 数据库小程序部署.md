## 第一阶段：本地环境搭建与联调

### Step 1 — 安装并启动 MongoDB

**方式A：本地安装（推荐开发阶段）**

1. 下载 MongoDB Community Edition：https://www.mongodb.com/try/download/community
2. 安装时勾选「Install MongoDB as a Service」，服务会自动启动
3. 验证是否启动成功：

```bash
mongosh --eval "db.adminCommand('ping')"
```

返回 `{ ok: 1 }` 即表示正常，`.env` 保持默认：

```
MONGODB_URI=mongodb://localhost:27017/shehong_jianshe
```

**方式B：使用 MongoDB Atlas 云数据库**

1. 注册：https://cloud.mongodb.com
2. 创建免费集群（M0 Shared）
3. 创建数据库用户，设置网络访问白名单（允许所有IP：0.0.0.0/0）
4. 复制连接字符串，填入 `.env`：

```
MONGODB_URI=mongodb+srv://用户名:密码@cluster0.xxxxx.mongodb.net/shehong_jianshe
```

### Step 3 — 初始化管理员账号

首个管理员无法通过接口创建（接口本身需要 ADMIN 权限），需直接用 mongosh 插入：

```bash
mongosh
```

```js
use shehong_jianshe

db.users.insertOne({
  name: "超级管理员",
  phone: "你的手机号（11位）",
  roles: ["ADMIN"],
  area: "",
  gridName: "",
  active: true,
  createdAt: new Date(),
  updatedAt: new Date()
})
```

插入成功后，用该手机号登录小程序即可获得管理员权限。

---

### Step 4 — 录入人员配置数据

登录管理员账号后，调用接口逐条录入17个区域的全部人员。

每条人员数据格式：

```json
{
  "name": "张三",
  "phone": "138xxxxxxxx",
  "roles": ["DESIGN"],
  "area": "射洪太和东",
  "gridName": "太和东服务中心",
  "feishuId": "ou_xxxxxxxx"
}
```

角色取值说明：

| 角色值 | 说明 |
|---|---|
| `FRONTLINE` | 一线人员 |
| `GRID_MANAGER` | 网格经理 |
| `NETWORK_MANAGER` | 网络支撑经理 |
| `DESIGN` | 设计单位 |
| `CONSTRUCTION` | 施工单位 |
| `SUPERVISOR` | 监理单位 |
| `ADMIN` | 管理员 |

`feishuId` 从飞书管理后台（admin.feishu.cn）通讯录中查询，格式为 `ou_xxxxxxxx`。

编辑 `miniprogram/app.js`：


### Step 9 — 在微信公众平台配置服务器域名（上线前）

微信公众平台 → 开发 → 开发管理 → 服务器域名，添加：

```
request 合法域名：       https://你的域名.com
uploadFile 合法域名：    https://你的域名.com
downloadFile 合法域名：  https://你的域名.com
```

> 本地联调阶段：微信开发者工具中勾选「不校验合法域名、web-view、TLS版本以及HTTPS证书」即可跳过此步。

## 第三阶段：本地端到端测试

### Step 10 — 用微信开发者工具打开小程序

1. 下载微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 导入项目：选择 `D:/AI/应用开发/Shehong_jianshe/miniprogram`
3. 顶部菜单勾选「详情」→「本地设置」→「不校验合法域名」
4. 按以下流程逐一测试：

**核心流程测试**

```
① 登录（手机号认证）
     → 验证：手机号在人员表中 → 登录成功，跳转角色选择页
     → 验证：手机号不在人员表 → 提示"未授权"

② 一线人员录入需求
     → 验证：表单提交成功，生成工单编号
     → 验证：飞书群收到【新需求】消息（场景1）

③ 设计单位提交查勘
     → 验证：需求状态变为"待施工"
     → 验证：飞书群收到【设计完成】消息（场景3）

④ 施工单位开始施工 → 提交施工结果
     → 验证：需求状态变为"已开通"
     → 验证：飞书群收到【施工完成】消息（场景4）

⑤ 设计单位驳回需求
     → 验证：需求状态变为"已驳回"
     → 验证：飞书群收到【驳回】消息（场景2）

⑥ 一线人员重新提交被驳回需求
     → 验证：状态重置为"设计中"，设计工期重新计算

⑦ 超时预警测试（修改数据库时间模拟）
     → 将某需求的 designAssignTime 改为36小时前，触发定时检查
     → 验证：飞书群收到【设计即将超时】预警（场景5）
```

## 第四阶段：生产环境部署

### Step 11 — 准备服务器

需要一台带公网 IP 的云服务器（腾讯云/阿里云 2核4G 起步），并完成：

1. 安装 Node.js 18+：

```bash
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs
```

2. 安装 MongoDB：

```bash
# Ubuntu 示例
wget -qO - https://www.mongodb.org/static/pgp/server-7.0.asc | sudo apt-key add -
echo "deb [ arch=amd64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/7.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-7.0.list
sudo apt-get update && sudo apt-get install -y mongodb-org
sudo systemctl start mongod && sudo systemctl enable mongod
```

3. 安装 PM2（进程守护）：

```bash
npm install -g pm2
```

4. 安装 Nginx：

```bash
sudo apt-get install -y nginx
```

---

### Step 12 — 申请 HTTPS 证书

微信小程序强制要求 HTTPS，推荐使用免费证书：

1. 在云服务商控制台申请免费 SSL 证书（腾讯云/阿里云均提供）
2. 下载证书文件，上传到服务器

---

### Step 13 — 配置 Nginx 反向代理

创建配置文件 `/etc/nginx/sites-available/shehong`：

```nginx
server {
    listen 80;
    server_name 你的域名.com;
    return 301 https://$host$request_uri;
}

server {
    listen 443 ssl;
    server_name 你的域名.com;

    ssl_certificate     /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;

    # 静态资源（上传文件）
    location /uploads/ {
        alias /home/app/shehong_jianshe/server/uploads/;
    }

    # API 反向代理
    location / {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
}
```

```bash
sudo ln -s /etc/nginx/sites-available/shehong /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

---

### Step 14 — 上传代码并启动服务

```bash
# 上传 server 目录到服务器（scp 或 git clone）
cd /home/app/shehong_jianshe/server

# 安装依赖
npm install --production

# 创建 uploads 目录
mkdir -p uploads

# 用 PM2 启动并设置开机自启
pm2 start src/app.js --name shehong-jianshe
pm2 save
pm2 startup
```

3. 微信公众平台 → 管理 → 版本管理 → 提交审核
4. 审核通过（通常1~3个工作日）→ 发布上线
