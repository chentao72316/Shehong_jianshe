# 图片资源说明

本目录存放小程序所需图片资源。

## tab 图标（miniprogram/images/tab/）

tabBar 需要以下图标，请提供 81x81px 的 PNG 图片：

| 文件名 | 用途 | 颜色建议 |
|--------|------|---------|
| home.png | 首页默认图标 | 灰色 #8C8C8C |
| home-active.png | 首页选中图标 | 主色 #1E6F3F |
| demand.png | 需求默认图标 | 灰色 #8C8C8C |
| demand-active.png | 需求选中图标 | 主色 #1E6F3F |
| message.png | 消息默认图标 | 灰色 #8C8C8C |
| message-active.png | 消息选中图标 | 主色 #1E6F3F |
| profile.png | 我的默认图标 | 灰色 #8C8C8C |
| profile-active.png | 我的选中图标 | 主色 #1E6F3F |

## 图标推荐来源

- 阿里巴巴矢量图标库（iconfont.cn）
- 微信小程序官方图标集
- 项目 UI 设计稿导出

## 注意

- tabBar 图标大小建议 81x81px
- 使用 PNG 格式，支持透明背景
- 文件大小建议不超过 40KB
